RunnersHigh

RunnersHigh is actually a jump&run game for Android, which was created by several students of the Fh-Salzburg in 2011. I was interested to rewrite the game in javascript using HTML5 canvas ... just for fun. Some things are still missing like the obstacles. However, the main issue remains the performance.

The entire code is available at github.
